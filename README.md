# Car Price Prediction with Machine Learning

## Overview
This project predicts the selling price of a car using Machine Learning techniques. The model is trained on various car features such as present price, kilometers driven, fuel type, transmission type, owner details, and car age.

## Objectives
- Perform data preprocessing and cleaning.
- Apply feature engineering techniques.
- Train a Linear Regression model.
- Evaluate model performance.
- Predict car selling prices accurately.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset Features
- Car_Name
- Year
- Selling_Price
- Present_Price
- Kms_Driven
- Fuel_Type
- Selling_type
- Transmission
- Owner

## Project Workflow
1. Import required libraries.
2. Load the dataset.
3. Handle missing values.
4. Encode categorical features.
5. Create a new feature: Car Age.
6. Split data into training and testing sets.
7. Train a Linear Regression model.
8. Evaluate model performance.
9. Visualize results.
10. Save the trained model.

## Model Used
- Linear Regression

## Results
The model was successfully trained and used to predict car prices based on input features.
